# xcode

### save all
`cmd + alt + s`

### save copy as
`cmd + alt + shift + s`

## compilation

### clean
`cmd + shift + k`

### build
`cmd + b`

## debug

### run
`cmd + r`

### toggle breakpoint
`cmd + \`

### continue execution
`cmd + ctrl + y`

### continue to current line
`cmd + ctrl + c`

### step into
`F7`

### step over
`F6`

### step out
`F8`

### toggle debug window
`cmd + shift + y`
