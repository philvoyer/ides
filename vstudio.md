# Visual Studio

## file

### save all
`ctrl + shift + s`

## compilation

### build
`ctrl + shift + b`

## debug

### start without debugging
`ctrl + F5`

### start debugging
`F5`

### toggle breakpoint
`F9`

### disable breakpoint
`ctrl + F9`

### step over
`F10`

### step into
`F11`

### step out
`shift + F11`

### Run to cursor
`ctrl + F10`

### continue
`F5`

## navigation

### navigate windows

`ctrl + TAB`

### navigate back / forward
`ctrl + -` / `ctrl + shift + -`

### navigate last change

`ctrl + shift + backspace`

### go to line
`ctrl + g`

### go to file
`ctrl + shift + t`

### bookmart set/unset
`ctrl + k ctrl + k`

### bookmart next/previous
`ctrl + k ctrl + n` / `ctrl + k ctrl + p`

## outlining

### toggle collapse / expand
`ctrl + m ctrl + m`

### collapse to definitions
`ctrl + m ctrl + o`

### toggle all outlining
`ctrl + m ctrl + l`

### stop outlining
`ctrl + m ctrl + p`

`ctrl + r ctrl + w`

### duplicate

### expand selection
`shift + alt + =`

### contract selection
`shift + alt + -`

### add caret

### rename
`ctrl + r ctrl + r`
