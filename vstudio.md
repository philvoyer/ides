## file

`C-S-s`

## compilation

### build
`C-S-b`

## debug

### start without debugging
`C-F5`

### start debugging
`F5`

### toggle breakpoint
`F9`

### step over
`F10`

### step into
`F11`

### step out
`S-F11`

### continue
`F5`

`C-g`

`C-k C-k`

`C-k C-n` / `C-k C-p`
