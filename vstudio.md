# Visual Studio

## file 

### save all
`C-S-s`

## compilation

### build
`C-S-b`

## debug

### start without debugging
`C-F5`

### start debugging
`F5`

### toggle breakpoint
`F9`

### disable breakpoint
`C-F9`

### step over
`F10`

### step into
`F11`

### step out
`S-F11`

### Run to cursor
`C-F10`

### continue
`F5`

## navigation

### navigate windows

`C-TAB`

### navigate back / forward
`C--` / `C-S--`

### go to line
`C-g`

### go to file
`C-S-t`

### bookmart set/unset
`C-k C-k`

### bookmart next/previous
`C-k C-n` / `C-k C-p`

### collapse to definitions
`C-m C-o`

### toggle all outlining
`C-m C-l`

### stop outlining
`C-m C-p`
